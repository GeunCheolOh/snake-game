# Snake Game

A simple and classic Snake Game implemented in HTML, CSS, and JavaScript.

## How to Play
- Use the arrow keys on your keyboard to control the snake.
- Eat the red food blocks to grow longer and increase your score.
- Avoid hitting the walls or your own body, or the game will be over.
- You can restart the game anytime by clicking the '재시작' (Restart) button.
- Adjust the speed of the game using the speed selector.

## Features
- Responsive and simple UI
- Local ranking system (Top 5 scores saved in your browser)
- Multiple speed options

## Getting Started
1. Clone or download this repository.
2. Open `index.html` in your web browser.

Enjoy the game!
